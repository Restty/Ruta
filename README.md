Instalación
-----------

### Requirements

- PHP 5.4.*

### Github

    cd /var/www
    git clone https://github.com/mostofreddy/ruta.git
    cd ruta
    composer install

### Composer

    "require": {
        "php": ">=5.4.0",
        "rocketcode/ruta": "*",
    }
